# مشروع إدارة المدرسة

## وصف المشروع

هذا المشروع هو تطبيق بسيط لإدارة قاعدة بيانات مدرسية باستخدام Node.js وSQLite. يتيح للمستخدمين إدارة معلومات الطلاب والدروس من خلال واجهة سطر الأوامر.

## الخصائص المتاحة

- **إضافة طالب جديد**: إدخال معلومات الطالب مثل الرقم، الاسم الأول، الاسم الأخير، العمر، الفصل، تاريخ التسجيل، والدروس المسجلة.
- **حذف طالب**: حذف طالب بناءً على رقمه.
- **تحديث معلومات طالب**: تعديل معلومات طالب موجود.
- **عرض معلومات طالب**: عرض تفاصيل طالب محدد.

## المتطلبات

- Node.js
- npm
- sqlite3

## الأوامر اللازمة لتشغيل المشروع

```
node script.js
```

سيتم إنشاء قاعدة البيانات تلقائيًا عند التشغيل الأول.
